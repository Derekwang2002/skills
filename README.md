# Skills

Reusable Agent Skills maintained by Derek Wang.

## Available Skills

- [`agent-eval`](agent-eval/): review local git diffs and run bounded reviewer/fixer workflows for Agent Skills.
- [`sync-post-localizations`](sync-post-localizations/): synchronize changed Markdown posts across every supported locale and repair coupled site behavior.

Reader-facing introductions live in [`docs/`](docs/).

## Install

Install a skill globally with the Skills CLI:

```bash
npx skills add Derekwang2002/skills@sync-post-localizations -g -y
```
